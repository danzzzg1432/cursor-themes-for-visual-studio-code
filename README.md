# Cursor Themes for Visual Studio Code

This repository contains Cursor's built-in default themes exported from `Cursor.app` for local use in Visual Studio Code on macOS.

Included themes:

- Cursor Dark
- Cursor Dark Midnight
- Cursor Dark High Contrast
- Cursor Light

Files:

- `package.json` defines the VS Code theme extension manifest
- `themes/` contains the original exported Cursor theme JSON files
- `cursor-built-in-themes-0.0.1.vsix` is the packaged VS Code extension

Install:

```bash
code --install-extension cursor-built-in-themes-0.0.1.vsix
```

Or install the `.vsix` from the Visual Studio Code Extensions panel via `Install from VSIX...`.
